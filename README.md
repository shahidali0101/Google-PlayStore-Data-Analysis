# Google-PlayStore-Data-Analysis
---

## 📌 Project Overview

This project analyzes the Google Play Store dataset to uncover trends in app categories, installs, ratings, and user engagement. It includes data cleaning, feature engineering, and exploratory data analysis (EDA) to extract meaningful insights

---

## 📊 Dataset

* **Total Apps**: 10,841
* **Columns**: 20 (App, Category, Rating, Reviews, Size, Installs, Price, etc.)

---

## 🛠️ Tech Stack

* **Python**
* Pandas, NumPy
* Matplotlib, Seaborn

**Skills:** Data Cleaning | EDA | Feature Engineering

---

## 🔄 Project Workflow

### 1. Data Cleaning

* Handled missing values
* Cleaned columns like Installs, Price, Size
* Converted data types

### 2. Feature Engineering

* Converted string data to numeric
* Parsed **Last Updated** into Day, Month, Year

### 3. Exploratory Data Analysis (EDA)

* Category-wise analysis
* Rating distribution
* Installs and reviews analysis
* Price vs rating comparison

## 🔍 Key Insights

* Games category dominates with **35B+ installs**
* Most apps are free, indicating revenue via ads or in-app purchases
* Top apps include **Subway Surfers** and **Hangouts**
* Around **269 apps have a perfect 5.0 rating**
* Higher installs are strongly related to higher user reviews
